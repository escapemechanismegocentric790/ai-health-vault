# Prompt: 生成日常健康管理方案

## 使用方式

把个人体检档案和就医记录发给 AI，让它生成个性化的日常管理方案。

## Prompt

```
请基于我的健康档案，生成一份个性化的日常健康管理方案：

### 1. 饮食管理
- 基于我的体检数据（血糖、血脂、肝肾功能等），列出：
  - 建议多吃的食物（具体到食物名，不要泛泛说"多吃蔬菜"）
  - 需要少吃/避免的食物
  - 每日饮水建议
- 如果有既往病史（如胆囊切除、糖尿病等），针对性调整

### 2. 运动建议
- 基于我的运动数据和身体状况：
  - 推荐的运动类型和频率
  - 需要避免的运动（如有关节问题）
  - 每日步数目标
  - 久坐提醒策略

### 3. 用药/补剂管理
- 当前用药的注意事项汇总
- 基于体检数据，是否需要补充营养素（如维生素D、钙等）
- 药物之间的相互作用提醒

### 4. 就医信号
- 列出需要立即就医的危险信号（基于我的既往史）
- 列出需要尽快预约复查的信号
- 日常自我监测要点

### 5. 睡眠建议
- 基于 Apple Watch 睡眠数据的改善建议

请用简洁的清单格式输出，方便我日常参考。
不需要长篇解释，每条建议一句话。
```

## 给家人的版本

```
请把上面的管理方案简化为给 [爸爸/妈妈] 的版本：
- 只保留最重要的 5-8 条
- 用口语化表达
- 每条配一个具体的行动（不要说"注意饮食"，要说"早餐别吃油条改吃鸡蛋"）
```
