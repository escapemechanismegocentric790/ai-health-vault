# Prompt: 药盒/药瓶识别

## 使用方式

拍药盒或药瓶照片，连同下面的 Prompt 一起发给 AI。

## Prompt

```
请识别这张照片中的药物信息，提取：

1. 药物名称（通用名 + 商品名）
2. 规格/剂量
3. 用法用量（说明书建议）
4. 主要功效
5. 常见副作用
6. 注意事项（饮食禁忌、药物相互作用等）

输出为 Markdown 格式，方便存入用药记录。

如果我提供了当前用药清单，请检查是否存在药物相互作用。
```

## 批量识别

多种药一起拍：

```
照片中有多种药物，请逐一识别并列表。
同时检查这些药物之间是否有相互作用或重复成分。
```
