# Prompt: 就医前准备

## 使用方式

告诉 AI 你要看什么科，AI 会基于你的历史档案生成就医清单。

## Prompt

```
我要去看 [XX科]，原因是 [简述症状/目的]。

请基于我的健康档案，帮我准备：

1. **历史摘要**（1 页纸，带给医生看）
   - 相关既往史
   - 相关用药
   - 相关检查结果和趋势

2. **本次应关注的指标**
   - 基于历史数据，哪些项目需要重点检查

3. **建议追问医生的问题**（5-8 个）
   - 围绕我的具体情况，不是泛泛的问题

4. **就医后记录模板**
   - 方便我在看完病后快速记录医生的结论和建议
```

## 就医后更新

```
刚看完 [XX科]，医生的结论是：[粘贴/描述]。
请帮我：
1. 更新我的就医记录
2. 更新体检档案中的相关信息
3. 如果有新的复查计划，加入提醒
```
