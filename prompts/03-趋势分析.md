# Prompt: 健康数据趋势分析

## 使用方式

把体检档案.md 的内容发给 AI，请求分析。

## Prompt

```
以下是我的体检档案（包含多次体检数据和 Apple Watch 持续监测数据）。

请帮我做一次全面的趋势分析：

1. **持续改善的指标**（标注 ✅）
2. **持续恶化的指标**（标注 🔴，解释可能原因）
3. **波动较大的指标**（标注 ⚠️，分析可能的影响因素）
4. **缺失数据**（哪些重要指标没有记录，建议下次体检补充）

最后给出：
- 最需要优先关注的 3 个健康问题
- 每个问题的具体行动建议（生活方式调整 + 是否需要就医）
- 下次体检建议重点检查的项目

请用非专业语言解释，让没有医学背景的人也能看懂。
```
