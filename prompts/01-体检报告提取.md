# Prompt: 体检报告提取

## 使用方式

拍照或上传体检报告 PDF，然后把下面的 Prompt 和照片一起发给 AI。

## Prompt

```
请帮我提取这份体检报告的数据，按以下格式输出：

1. 基本信息（姓名、日期、医院）
2. 所有检查项目，每项包含：
   - 项目名称
   - 检查结果（数值）
   - 参考范围
   - 是否异常（✅ 正常 / ⚠️ 偏高 / ⚠️ 偏低 / 🔴 异常）

输出为 Markdown 表格格式，方便我直接粘贴到 Obsidian 笔记里。

如果有上一次的体检数据（我会提供），请在表格中增加「趋势」列，标注 ↑升高 / ↓降低 / →稳定。

最后，请单独列出：
- 需要关注的异常项（解释每项异常的含义和可能原因）
- 建议的复查项目和时间
```

## 进阶用法

如果你有多次体检报告，可以追加：

```
这是我最近 3 次体检的数据（已在体检档案.md 中）。
请对比趋势，找出持续恶化的指标，并给出建议。
```

## 注意事项

- 拍照时确保文字清晰，光线充足
- 多页报告可以分多张图发送
- AI 可能会识别错个别数字，建议核对关键指标（血糖、血脂、肝肾功能）
