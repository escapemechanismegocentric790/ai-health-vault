# Prompt: 转成给父母看的版本

## 使用方式

把任意分析结果发给 AI，让它转成口语化的微信消息。

## Prompt

```
请把上面的分析结果转成微信消息格式，发给我 [爸/妈]：

要求：
1. 用口语化的中文，像跟长辈聊天一样
2. 不用专业术语，如果必须用就加括号解释
3. 重点标出需要注意的事项
4. 如果有需要他们做的事（比如复查、调整用药），明确写出时间和地点
5. 语气温和但不啰嗦，控制在微信一屏能看完的长度
6. 最后加一句关心的话

不要用 Markdown 格式，纯文字就好。
```

## 示例输出

```
妈，上次体检报告我帮你看了一下：

血糖 6.3，比上次高了一点点，还在正常范围但要注意了。少吃甜的，米饭也别吃太多。

维生素D 偏低，这个跟晒太阳少有关系。可以每天出去走走晒半小时太阳，或者吃维生素D的补充剂（药店有卖的，一天一粒就行）。

血脂四项都正常，这个不用担心。

下次复查建议 6 个月后再查一次血糖，到时候我提醒你。

注意身体，有不舒服随时跟我说 ❤️
```
